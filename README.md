# FUTURE_FS_01
Portfolio-website

This is my personal portfolio website created as a 3rd year Computer Engineering student who is passionate about Data Science, Web Development, and AI projects. The purpose of this website is to showcase my profile, highlight my academic and personal projects, and provide a contact form so that recruiters or collaborators can easily reach me. The website includes sections such as About Me, Projects, and Contact. It is built using HTML, CSS, and JavaScript for the frontend, and Node.js with Express and Nodemailer for the backend to handle email functionality. The contact form securely sends messages using Gmail SMTP with an App Password. To run this project, clone the repository, install the required dependencies using npm install, create a .env file with your email credentials, and start the server using node server.js. Then open http://localhost:5000
 in your browser.
